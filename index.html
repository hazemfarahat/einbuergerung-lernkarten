📲 Als App installieren

Installiere diese App auf deinem Gerät für schnellen Zugriff und Offline-Nutzung!

📱 iPhone / iPad (Safari)

  1. Tippe auf das Teilen-Symbol (Viereck mit Pfeil nach oben) in der Safari-Leiste
  2. Scrolle nach unten und wähle "Zum Home-Bildschirm"
  3. Tippe auf "Hinzufügen" oben rechts
  4. Die App erscheint auf deinem Home-Bildschirm 🎉

🤖 Android (Chrome)

  1. Tippe auf das Menü (drei Punkte) oben rechts
  2. Wähle "App installieren" oder "Zum Startbildschirm hinzufügen"
  3. Bestätige mit "Installieren"
  4. Die App erscheint in deiner App-Liste 🎉

💻 Desktop (Chrome / Edge)

  1. Klicke auf das Installieren-Symbol in der Adressleiste (rechts)
  2. Oder: Menü → "Installieren..."
  3. Bestätige die Installation
  4. Die App öffnet sich in einem eigenen Fenster 🎉

✨ Vorteile

  • 🚀 Schneller Zugriff wie eine native App
  • 📴 Funktioniert auch offline
  • 💾 Dein Lernfortschritt wird gespeichert
  • 🔔 Keine Installation aus App Store nötig
